## How does GiftCardify work?

Gift card form includes:
```
Amount:
  Preset prices(For example, 100, 250, 500) and custom price
Receiver:
  First name
  Last name
  Email(and Email confirm)
Sender:
  Sender name
Gift message
Shipping date
```
